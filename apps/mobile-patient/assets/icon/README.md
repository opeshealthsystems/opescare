# App Icon

Place a 1024×1024 PNG named `icon.png` in this directory.

Design spec:
- Background: `#1565C0` (Clinical Blue, matching `AppColors.primary500`)
- Foreground: White heart icon centered at ~40% canvas size
- Matches the brand mark used in `LoginScreen`

After placing the file, run:
```
dart run flutter_launcher_icons
```
