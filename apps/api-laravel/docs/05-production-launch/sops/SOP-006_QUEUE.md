# SOP-006 — Queue & Patient Flow

**SOP Number:** SOP-006 | **Version:** 1.0 | **Owner:** Receptionist / Queue Manager

---

## 1. Purpose
Manage patient queues efficiently and fairly, respecting clinical priority.

## 2. Check-In Procedure
1. Verify patient identity (SOP-002).
2. Record reason for visit.
3. Assign priority (P1-P5):
   - P1: Life-threatening emergency → triage immediately (SOP-007)
   - P2: Urgent (severe pain, significant distress)
   - P3: Normal clinic visit
   - P4: Routine review
   - P5: Admin/documentation only
4. Issue queue ticket.

## 3. Calling Patients
1. Call by name + ticket number.
2. Patient acknowledged: start service.
3. No response after 2 calls: mark waiting/no-response; retry in 5 min.

## 4. Priority Escalation
Any staff member may escalate a patient's priority if clinical condition changes.
