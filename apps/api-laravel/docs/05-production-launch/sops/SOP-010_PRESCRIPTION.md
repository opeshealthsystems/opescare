# SOP-010 — Prescriptions

**SOP Number:** SOP-010 | **Version:** 1.0 | **Owner:** Prescribing Clinician

---

## 1. Pre-Prescribing
Always review patient allergies and active medications before prescribing.

## 2. Issue Prescription in OpesCare
1. Select drug, enter dose, frequency, duration, route, indication.
2. System runs CDSS checks: allergy, drug-drug interaction, age, pregnancy, Beers Criteria.
3. Review all alerts.

## 3. CDSS Alert Handling
Alerts are decision-support only. If overriding: document reason (min 10 chars) and category.
Never override a critical allergy alert without documented clinical justification.

## 4. Authorise Prescription
Review summary, sign electronically. Prescription sent to pharmacy queue.

## 5. Controlled Substances
Requires dual authorisation. Quantity and batch number recorded. Pharmacist confirms on receipt.
