# SOP-009 — Laboratory Requests & Results

**SOP Number:** SOP-009 | **Version:** 1.0 | **Owner:** Lab Technician / Requesting Clinician

---

## 1. Lab Request Procedure
1. Clinician issues request via OpesCare consultation note, selecting test(s) and adding clinical indication.
2. Lab receives request in OpesCare worklist.
3. Verify patient identity before specimen collection.
4. Label specimen with patient Health ID + barcode. Record collection time.

## 2. Processing & Results
1. Process per lab protocol.
2. Enter results in OpesCare.
3. Critical values: flag immediately and phone requesting clinician (document call).

## 3. Result Return
Results appear in clinician worklist. CDSS lab alert rules may trigger (review each).
Clinician acknowledges result and records action taken.

## 4. Critical Value Notification
If result meets critical threshold: OpesCare triggers alert + lab staff phones clinician directly. Document call in OpesCare.
