# SOP-013 — Insurance Claims & Preauthorisation

**SOP Number:** SOP-013 | **Version:** 1.0 | **Owner:** Insurance Desk / Claims Officer

---

## 1. Verify Insurance
Confirm NHIS/HMO number in OpesCare. Verify policy is active. Note covered services and limits.

## 2. Preauthorisation
If procedure requires preauth: submit via OpesCare (Insurance > Preauth). Include clinical indication. Await approval.
Record approval code in OpesCare.

## 3. Claims Submission
1. Verify services match preauth approval.
2. Attach documents (clinical notes, labs, discharge summary).
3. Submit via OpesCare (Insurance > Claims).
4. Track status. Respond to insurer queries within 5 business days.

## 4. Rejected Claims
Review reason. Correct errors. Resubmit with supporting documentation.
Escalate repeated rejections to supervisor.
