# SOP-001 — Patient Registration

**SOP Number:** SOP-001 | **Version:** 1.0 | **Owner:** Registration Clerk

---

## 1. Purpose
Standardise new patient registration in OpesCare ensuring accurate data capture, Health ID generation, and informed consent.

## 2. When to Use
Patient presents for the first time with no existing OpesCare Health ID.

## 3. Procedure

### Step 1 — Check for Existing Record
Search by name, phone, or existing ID. If found: update, do not duplicate.

### Step 2 — Capture Demographics
Required: full name, date of birth, gender, phone, address.
Recommended: NHIS/HMO number, next of kin, allergies.

### Step 3 — Obtain Consent
Present Patient Consent Policy (SOP-003). Record acceptance in OpesCare.

### Step 4 — Generate Health ID
OpesCare auto-generates Health ID on save. Print QR code for patient.

### Step 5 — Queue or Appointment
Check patient in to queue if walk-in, or book appointment.

## 4. Quality Checks
- No duplicate records
- Phone number verified
- Consent acceptance recorded
- Health ID card given to patient
