# SOP-002 — Health ID Verification

**SOP Number:** SOP-002 | **Version:** 1.0 | **Owner:** Any clinical/reception staff

---

## 1. Purpose
Verify patient identity using OpesCare Health ID before accessing records or performing clinical actions.

## 2. When to Use
Before any clinical action, billing, document access, or consent to a procedure.

## 3. Procedure

### Step 1 — Obtain Health ID
Ask for card. If none: search by name + DOB. If scanning: scan QR code.

### Step 2 — Verify Identity
Compare name and DOB on screen with patient or ID document.

### Step 3 — Flag Discrepancies
If mismatch: do NOT proceed. Escalate to supervisor and document.

## 4. Emergency Exception
Unconscious patient: use emergency access protocol (SOP-004).
