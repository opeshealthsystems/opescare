# SOP-007 — Triage

**SOP Number:** SOP-007 | **Version:** 1.0 | **Owner:** Triage Nurse / Clinical Lead

---

## 1. Triage Categories

| Category | Colour | Response | Description |
|----------|--------|----------|-------------|
| P1 Immediate | Red | Now | Life-threatening |
| P2 Urgent | Orange | <10 min | Could deteriorate |
| P3 Semi-urgent | Yellow | <30 min | Needs care soon |
| P4 Non-urgent | Green | <60 min | Can safely wait |
| P5 Routine | Blue | As available | Scheduled/admin |

## 2. Triage Procedure (max 2 minutes)

1. Vital signs: BP, pulse, SpO2, temperature, RR.
2. AVPU assessment (Alert / Voice / Pain / Unresponsive).
3. Chief complaint (one sentence).
4. Visible bleeding or distress.
5. Assign category in OpesCare. Document findings.

## 3. P1 Escalation
Move to resuscitation bay immediately. Alert doctor. Initiate emergency protocol.

## 4. CDSS Integration
CDSS alerts during triage are decision-support only. Clinical judgment prevails.
