# SOP-018 — Incident Reporting

**SOP Number:** SOP-018 | **Version:** 1.0 | **Owner:** Compliance Officer

---

## 1. Incident Categories

| Category | Examples |
|----------|---------|
| Security | Unauthorised access, data breach, device theft |
| Clinical | Wrong patient, wrong drug, near miss |
| Operational | Downtime, data loss, import error |
| Compliance | NDPR violation, consent failure |

## 2. Reporting Procedure
1. Any discoverer reports within 1 hour to supervisor + Compliance Officer (data/security) or Clinical Lead (clinical).
2. Log in OpesCare Support > Incidents: date/time, category, description, affected parties, immediate actions.
3. Contain: take immediate steps to stop ongoing harm.
4. Investigate: root cause analysis within 5 business days.
5. Notify: data breach affecting patients — notify NITDA within 72 hours (NDPR).
6. Remediate: implement corrective actions. Review at monthly governance meeting. Update SOPs if process gap identified.
