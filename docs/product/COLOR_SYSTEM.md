# OpesCare Color System

## 1. Design Philosophy
The OpesCare color system is designed for clinical precision, trustworthiness, and accessibility. It prioritizes readability and clear status communication.

## 2. Core Tokens

### Primary Blue (Brand & Action)
- `primary-50`: `#E8F2FA`
- `primary-100`: `#D4E8F6`
- `primary-600`: `#0F4C81` (Primary)
- `primary-700`: `#0A3F6B`
- `primary-800`: `#0A355C`

### Clinical Teal (Verified & Safe)
- `clinical-50`: `#E6F7F5`
- `clinical-100`: `#CCEFEB`
- `clinical-600`: `#0F766E`
- `clinical-700`: `#0B5F59`

### UI Surfaces
- `background-default`: `#F7FAFC`
- `surface-default`: `#FFFFFF`
- `surface-muted`: `#F1F5F9`
- `surface-subtle`: `#F8FAFC`
- `border-default`: `#E2E8F0`
- `border-strong`: `#CBD5E1`

### Status Tokens
- **Success**: `#15803D` (600), `#166534` (700)
- **Warning**: `#B45309` (600), `#92400E` (700)
- **Danger**: `#B91C1C` (700), `#991B1B` (800)
- **Critical**: `#991B1B` (700), `#7F1D1D` (900)
- **Info**: `#0369A1` (700), `#075985` (800)
- **System (Purple)**: `#6D28D9` (700), `#5B21B6` (800)

## 3. Accessibility
All text combinations must meet WCAG AA standards. Critical information must never rely on color alone.
