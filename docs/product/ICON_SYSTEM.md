# OpesCare Icon System

## 1. Icon Library
OpesCare uses **Lucide Icons** as the primary icon library for a clean, professional, and clinical look.

## 2. Icon Standards
- **Stroke Width**: 1.75px to 2px
- **Default Size**: 18px to 20px
- **Dashboard Card Size**: 22px to 26px
- **Empty State Size**: 36px to 48px
- **Accessibility**: Icons must always be paired with text labels in critical workflows.

## 3. Module Mapping

### Core Platform
- Dashboard → `LayoutDashboard`
- Search → `Search`
- Settings → `Settings`
- Notifications → `Bell`
- User Profile → `User`
- Facility Selector → `Building2`
- Help → `CircleHelp`
- Logout → `LogOut`

### Patient Identity
- Patients → `Users`
- Patient Profile → `UserRound`
- Health ID → `IdCard`
- Verified Patient → `BadgeCheck`

### Clinical Operations
- Consultation → `Stethoscope`
- Triage → `Activity`
- Vitals → `HeartPulse`
- Laboratory → `FlaskConical`
- Pharmacy → `Pill`
- Medication → `Capsules`
- Blood Bank → `Droplets`

### Institutional Admin
- HR / Staff → `Users`
- Inventory → `Boxes`
- Assets → `Package`
- Reports → `FileText`
- Integration → `Cable`

## 4. Implementation Rule
Do not use emojis or inconsistent icon packs. If a specific mapping is missing, use the closest Lucide equivalent.
